# 一键生成APK - 超简单方法

## 🎯 最简单的方法 (无需安装任何软件!)

### 方案: 使用在线APK生成器

**步骤:**

1. **准备文件**
   - 文件已打包好: `E:\lottery-betting-tool\lottery-app.zip`

2. **访问在线工具** (任选一个):
   
   **推荐1: AppsGeyser**
   - 网址: https://appsgeyser.com/
   - 免费
   - 不需要注册
   
   **推荐2: Appilix**
   - 网址: https://appilix.com/
   - 免费试用
   
   **推荐3: Web2Desk**
   - 网址: https://web2desk.com/

3. **上传文件**
   - 选择"Upload HTML Files"或"Local HTML"
   - 上传 `lottery-app.zip`
   - 或者解压后上传整个文件夹

4. **配置APP**
   - APP名称: 六合彩工具
   - 包名: com.lottery.tool
   - 版本: 1.0
   - 图标: (可选,可以跳过)

5. **生成APK**
   - 点击"Build APK"或"Generate"
   - 等待1-3分钟
   - 下载APK文件

6. **安装到手机**
   - 通过微信/QQ发送到手机
   - 或者上传到网盘下载
   - 手机点击安装

## 📱 更简单的替代方案

### 如果上面太麻烦,直接用这个:

**PWA (渐进式Web应用)**

1. **在电脑上:**
   - 打开Chrome浏览器
   - 访问: `file:///E:/lottery-betting-tool/index.html`

2. **创建快捷方式:**
   - Chrome右上角三个点
   - 更多工具 → 创建快捷方式
   - 勾选"作为窗口打开"
   - 点击"创建"

3. **发送到手机:**
   - 把整个 `lottery-betting-tool` 文件夹复制到手机
   - 手机用Chrome打开 `index.html`
   - 添加到主屏幕
