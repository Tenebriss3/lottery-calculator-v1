# 六合彩下注管理系统 - 使用说明

## 📱 快速开始

### 1. 打开系统
直接在浏览器中打开 `index.html` 文件即可使用。

**支持的浏览器:**
- Chrome 80+
- Safari 13+
- Firefox 75+
- Edge 80+

**支持的设备:**
- 📱 手机 (竖屏/横屏)
- 📟 平板 (竖屏/横屏)
- 💻 电脑

---

## 🎯 核心功能

### 1️⃣ 智能解析下注

**操作步骤:**
1. 在左侧"聊天记录粘贴区"粘贴客户的下注消息
2. 点击"🔍 智能解析"按钮
3. 右侧自动显示解析结果

**支持的格式:**

```
✅ 平特蛇2000
✅ 26.38.41各15米
✅ 新澳 26.38.41.30.20各15米
✅ 包猪各25
✅ 25.=50米
✅ 新奥： 特码01,02,06,07每个数5蚊
✅ 22.34.46.12各20
✅ 05.17.29.01各10
```

**金额单位说明:**
- `米` = 1000元
- `蚊/块/元` = 1元

**示例:**
- `15米` = 15000元
- `5蚊` = 5元

---

### 2️⃣ 查看解析结果

解析结果区域包含三个表格:

**🔢 号码统计**
- 显示每个号码出现的次数
- 显示每个号码的累计投注金额

**🐉 生肖投注**
- 显示所有生肖投注记录
- 显示对应的号码和金额

**📝 下注明细**
- 显示每一条下注记录
- 包含原始聊天记录

💡 **提示:** 点击"📋 解析结果"标题可以折叠/展开结果区

---

### 3️⃣ 计算盈亏

**操作步骤:**
1. 在底部"今日开奖号码"输入框中输入中奖号码
2. 用逗号或空格分隔多个号码
3. 点击"📊 计算盈亏"按钮
4. 系统自动显示:
   - 今日总投注
   - 需赔付总额
   - 今日盈亏 (绿色=盈利, 红色=亏损)

**赔率:** 固定 1:47

**示例输入:**
```
07,19,31,43
```
或
```
07 19 31 43
```

---

### 4️⃣ 导出数据

#### 导出Excel
点击"📥 导出Excel"按钮,自动生成包含三个工作表的Excel文件:

**Sheet1: 当日前台操作页**
- 完整的下注明细
- 号码统计表
- 今日盈亏核算结果

**Sheet2: 生肖配置后台**
- 十二生肖对应号码
- 相冲属相关系
- 赔率配置

**Sheet3: 全年历史归档总账**
- 每日盈亏汇总
- 历史开奖记录

#### 文本备份
点击"💾 文本备份"按钮,生成JSON格式的备份文件,方便数据恢复。

---

### 5️⃣ 导入数据

**操作步骤:**
1. 点击"📤 导入数据"按钮
2. 选择之前导出的Excel或JSON文件
3. 系统自动恢复数据

**用途:**
- 多设备间同步数据
- 数据备份恢复
- 查看历史记录

---

### 6️⃣ 归档数据

**操作步骤:**
1. 点击"📦 归档今日数据"按钮
2. 确认归档
3. 系统自动:
   - 将今日数据保存到历史记录
   - 清空当前页面
   - 第二天可以重新开始使用

**建议:** 每天结束后进行归档

---

## 💡 使用技巧

### 1. 批量粘贴
可以一次性粘贴多条下注记录,系统会自动逐行解析。

### 2. 实时保存
所有解析结果自动保存在浏览器本地,关闭页面后重新打开仍然保留。

### 3. 多设备同步
通过"导出Excel" → 传输到另一设备 → "导入数据"实现同步。

### 4. 定期备份
建议每天归档前导出Excel备份,防止数据丢失。

### 5. 快捷键
- 在开奖号码输入框按 `Enter` 键快速计算盈亏

---

## ⚠️ 注意事项

### 数据安全
- 数据存储在浏览器LocalStorage中
- 清除浏览器缓存会丢失数据
- 建议定期导出备份

### 存储限制
- LocalStorage容量约5-10MB
- 大量历史数据建议及时归档导出

### 离线使用
- 首次打开需要联网加载SheetJS库
- 之后可以离线使用(如果浏览器已缓存)
- 或者下载SheetJS库到本地libs目录

## 🔧 故障排除

### 问题1: 解析结果为空
**原因:** 聊天记录格式不符合规范
**解决:** 检查格式是否正确,参考上方"支持的格式"

### 问题2: 无法导出Excel
**原因:** SheetJS库未加载
**解决:** 检查网络连接,或下载库文件到本地

### 问题3: 数据丢失
**原因:** 清除了浏览器缓存
**解决:** 从备份文件导入恢复

### 问题4: 手机端显示异常
**原因:** 浏览器兼容性问题
**解决:** 使用Chrome或Safari最新版
