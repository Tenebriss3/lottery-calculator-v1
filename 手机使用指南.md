# 手机端使用指南

## 🚀 三种方案

### 方案1: GitHub Pages (最推荐⭐)

**优点:**
- ✅ 不需要安装
- ✅ 一个网址,所有设备都能访问
- ✅ 自动更新
- ✅ 完全免费

**缺点:**
- ⚠️ 需要首次联网
- ⚠️ 数据存在各自浏览器中

**操作步骤:**

#### 第1步: 注册GitHub
1. 访问 https://github.com
2. 点击"Sign up"
3. 填写邮箱、密码、用户名
4. 验证邮箱

#### 第2步: 创建仓库
1. 登录后,点击右上角"+" → "New repository"
2. Repository name: `lottery-tool`
3. 选择"Public"
4. 点击"Create repository"

#### 第3步: 上传文件
1. 点击"uploading an existing file"链接
2. 打开文件夹 `E:\lottery-betting-tool\`
3. 选中所有文件(Ctrl+A)
4. 拖拽到GitHub上传区域
5. 等待上传完成
6. 点击绿色按钮"Commit changes"

#### 第4步: 开启GitHub Pages
1. 点击仓库顶部的"Settings"标签
2. 左侧菜单找到"Pages"
3. 在"Build and deployment"下:
   - Source: 选择"Deploy from a branch"
   - Branch: 选择"main"
   - Folder: 选择"/ (root)"
4. 点击"Save"

#### 第5步: 获取网址
1. 等待1-2分钟
2. 刷新页面
3. 顶部会显示: 
   ```
   Your site is live at https://你的用户名.github.io/lottery-tool/
   ```
4. 复制这个网址

#### 第6步: 手机访问
1. 手机浏览器打开上面的网址
2. 添加到主屏幕(更像APP):
   - iPhone Safari: 点击分享按钮 → "添加到主屏幕"
   - Android Chrome: 点击右上角菜单 → "添加到主屏幕"

### 方案2: 局域网访问 (适合家庭/办公室)

**前提:** 电脑和手机在同一WiFi网络

**步骤:**

1. **在电脑上启动本地服务器**
   ```bash
   # 如果有Python
   cd E:\lottery-betting-tool
   python -m http.server 8080
   ```

2. **查看电脑IP地址**
   ```bash
   # Windows
   ipconfig
   
   # 找到 "IPv4 地址",例如: 192.168.1.100
   ```

3. **手机访问**
   ```
   手机浏览器打开: http://192.168.1.100:8080
   ```

**优点:**
- ✅ 完全离线
- ✅ 速度快

**缺点:**
- ❌ 电脑必须开机
- ❌ 必须在同一WiFi
- ❌ 每次都要启动服务器

### 方案3: 打包成Android APK (高级)

**工具推荐:**

#### 选项A: 使用Cordova (需要Node.js)
```bash
npm install -g cordova
cordova create lottery com.example.lottery "六合彩工具"
cd lottery
# 复制HTML文件到 www 目录
cordova platform add android
cordova build android
```

#### 选项B: 使用在线工具
1. 访问: https://websitetoapk.com/
2. 上传HTML文件
3. 生成APK
4. 安装到手机

#### 选项C: 使用HBuilder (国产,简单)
1. 下载: https://www.dcloud.io/hbuilderx.html
2. 新建项目 → 5+App
3. 复制HTML文件
4. 云打包 → 生成APK

- 不同浏览器数据不互通
- 建议定期导出Excel备份

## ❓ 常见问题

### Q: GitHub Pages收费吗?
A: 完全免费!

### Q: 需要联网吗?
A: 首次加载需要,之后可以离线使用(浏览器缓存)

### Q: 数据安全吗?
A: 数据保存在你的手机浏览器中,GitHub只托管HTML文件,看不到你的数据

### Q: 可以多人使用吗?
A: 可以!每个人访问同一个网址,但数据各自独立
