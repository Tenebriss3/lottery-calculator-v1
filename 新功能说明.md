# 新版使用说明 - 追加模式和编辑功能

## 🎯 核心改进

### 1. 解析预览 + 确认机制
**之前的问题:** 粘贴后直接录入,无法检查是否正确

**现在的流程:**
```
1. 粘贴聊天记录
2. 点击"🔍 智能解析"
3. 👁️ 显示解析预览 (左侧新增预览区)
4. 检查无误后,点击"✅ 确认添加"
5. 数据正式录入系统
```

**预览区显示:**
- 识别记录数
- 总金额
- 每条记录的详细信息(类型、号码、金额、原始文本)

---

### 2. 多次追加下注
**之前的限制:** 每次解析会覆盖之前的数据

**现在的功能:**
- ✅ 可以连续粘贴多次聊天记录
- ✅ 每次确认后自动追加到已有数据
- ✅ 不会覆盖之前的下注记录

**使用场景:**
```
第1次: 粘贴上午的聊天记录 → 解析 → 确认添加
第2次: 粘贴下午的聊天记录 → 解析 → 确认添加
第3次: 粘贴晚上的聊天记录 → 解析 → 确认添加

最终: 所有数据累积在一起,自动统计总额
```

---

### 3. 编辑和删除功能
**位置:** 右侧"已录入数据" → "下注明细"表格

**操作按钮:**
- ✏️ **编辑按钮** - 修改某条记录的金额
- 🗑️ **删除按钮** - 删除某条记录

**使用示例:**

#### 编辑金额
```
场景: 客户说"刚才那单改一下,改成30米"

操作:
1. 找到对应的记录
2. 点击 ✏️ 按钮
3. 输入新金额 (例如: 30000)
4. 系统自动重新计算统计
```

#### 删除记录
```
场景: 客户取消了下注

操作:
1. 找到要删除的记录
2. 点击 🗑️ 按钮
3. 确认删除
4. 系统自动更新统计
```

---

### 4. 清空全部
**位置:** 右侧"已录入数据"顶部

**功能:** 一键清空所有已录入的数据(危险操作,需确认)

## 💡 使用技巧

### 技巧1: 批量追加
```
可以一次性粘贴多条记录:

平特蛇2000
26.38.41各15米
包猪各25

→ 解析 → 确认添加

然后继续粘贴:

只删除那一条,其他保留
```

```
每次添加/编辑/删除后:
- 号码统计表自动更新
- 生肖投注表自动更新
- 总收款自动更新

无需手动刷新!
```

## ⚠️ 注意事项

### 1. 确认前可以修改
在点击"确认添加"之前,可以随时:
- 修改粘贴框的内容
- 重新解析
- 取消操作 (点击"清空输入")

### 2. 确认后可以编辑
点击"确认添加"后,数据正式录入:
- 可以通过 ✏️ 修改金额
- 可以通过 🗑️ 删除记录
- 可以继续追加新记录

### 3. 清空全部需谨慎
"🗑️ 清空全部"按钮会删除所有数据:
- 有二次确认提示
- 删除后无法恢复
- 建议先导出备份

### 4. 自动保存
所有操作都会自动保存到浏览器:
- 关闭页面后再打开,数据还在
- 但清除浏览器缓存会丢失
- 建议定期导出备份

## 🎉 优势总结

### 相比旧版:
✅ **更安全** - 预览后再确认,避免错误录入
✅ **更灵活** - 可以多次追加,不用一次性粘贴
✅ **更易用** - 随时编辑/删除,不用重来
✅ **更清晰** - 预览区和已录入区分开

### 核心价值:
- **减少错误** - 预览机制让你有机会检查
- **提高效率** - 不用删除重加,直接修改
- **方便管理** - 每条记录都可追溯
