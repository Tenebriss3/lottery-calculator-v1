# 六合彩下注管理系统 - 项目交付说明

## 📦 项目概述

这是一个**纯前端离线网页工具**,用于管理六合彩下注业务,支持在手机、平板、电脑上使用。

### 核心特性
✅ 智能解析聊天记录中的下注信息
✅ 自动统计号码出现次数和累计金额
✅ 自动计算每日盈亏 (赔率1:47)
✅ Excel导入导出 (三张Sheet完整结构)
✅ 文本备份恢复 (JSON格式)
✅ 深色专业主题界面
✅ 响应式设计 (手机/平板/电脑自适应)
✅ 完全离线可用 (首次加载需联网)
✅ 数据本地存储 (LocalStorage)

## 🎯 已实现的功能

### 1. 智能解析引擎 ✅
**支持的格式:**
- `平特蛇2000` → 生肖投注
- `26.38.41各15米` → 批量号码(米=1000)
- `包猪各25` → 生肖全包
- `25.=50米` → 单个号码
- `特码01,02,06...每个数5蚊` → 特码投注
- `新澳 26.38.41...各15米` → 带前缀的批量

**解析结果:**
- 每个号码的出现次数
- 每个号码的累计投注金额
- 生肖投注明细
- 完整下注记录列表

### 2. 盈亏核算 ✅
- 输入开奖号码自动计算
- 总收款统计
- 总赔付计算 (赔率1:47)
- 盈亏判断 (绿色=盈利, 红色=亏损)
- 详细中奖记录

### 3. 数据管理 ✅
- **自动保存**: 解析结果自动保存到浏览器LocalStorage
- **Excel导出**: 三个Sheet完整结构
  - Sheet1: 当日前台操作页
  - Sheet2: 生肖配置后台
  - Sheet3: 全年历史归档总账
- **文本备份**: JSON格式,方便恢复
- **数据导入**: 支持Excel和JSON格式
- **归档功能**: 每日数据归档到历史记录

### 4. 界面设计 ✅
- **深色主题**: 专业配色方案
- **响应式布局**:
  - 手机: 垂直堆叠布局
  - 平板: 左右分栏
  - 电脑: 最大宽度1200px居中
- **触摸优化**: 按钮最小44px,防止误触
- **可折叠面板**: 解析结果区可折叠,不占主视觉
- **大字体显示**: 核心数据醒目展示

### 5. 配置管理 ✅
- 十二生肖完整号码映射
- 相冲属相关系表
- 固定赔率1:47
- 金额单位转换 (米/蚊/块/元)

## 📱 多设备使用

### 数据同步方案
由于是纯前端工具,数据存储在各自浏览器的LocalStorage中,多设备同步需要:

1. **设备A**: 点击"📥 导出Excel"或"💾 文本备份"
2. **传输文件**: 通过微信/QQ/邮件等方式传输到设备B
3. **设备B**: 点击"📤 导入数据",选择传输过来的文件

### 推荐流程
- **每天结束**: 导出Excel备份 + 归档数据
- **换设备时**: 导入最新备份文件
- **定期清理**: 删除旧备份,保留最近7天

## ⚙️ 技术细节

### 技术栈
- **HTML5** - 语义化标签
- **CSS3** - Flexbox布局、CSS变量、媒体查询
- **Vanilla JavaScript** - ES6+类语法,无框架依赖
- **SheetJS (xlsx.js)** - Excel文件读写

### 浏览器兼容性
- Chrome 80+
- Safari 13+
- Firefox 75+
- Edge 80+

### 数据存储
- **LocalStorage容量**: 约5-10MB
- **存储内容**:
  - 当日下注数据
  - 历史归档记录
- **持久化**: 关闭浏览器后数据仍保留

### 性能指标
- 解析100条记录: < 1秒
- 页面加载: < 2秒
- 导出1000条记录: < 3秒

---

## 🔒 数据安全

### 注意事项
⚠️ **重要提醒:**
1. 数据仅存储在浏览器本地
2. 清除浏览器缓存会丢失数据
3. 建议每天导出备份
4. 不要在不信任的设备上保存敏感数据

### 隐私保护
- 所有数据仅在本地处理
- 不会上传到任何服务器
- 完全离线可用(除首次加载SheetJS库)

---

## 🐛 已知限制

1. **LocalStorage限制**: 大数据量可能超出存储上限
2. **首次加载**: 需要联网下载SheetJS库(可从CDN下载到本地)
3. **浏览器兼容**: 不支持IE11及以下版本
4. **文件格式**: 仅支持xlsx/xls/json格式导入

## 🎨 自定义扩展

### 修改赔率
编辑 `js/config.js`:
```javascript
PAYOUT_RATE: 47, // 改为你需要的赔率
```

### 修改配色
编辑 `css/style.css` 中的CSS变量:
```css
:root {
    --bg-primary: #1a1a2e;      /* 主背景色 */
    --accent-blue: #3742fa;     /* 主色调 */
    --accent-green: #00d9ff;    /* 盈利颜色 */
    --accent-red: #ff4757;      /* 亏损颜色 */
}
```

### 添加新的解析规则
编辑 `js/parser.js`,在 `parseLine()` 方法中添加新的解析逻辑。
