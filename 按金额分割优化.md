# 按金额模式分割 - 智能解析优化

## 🎯 核心改进

### 之前的逻辑:
```
按标点符号分割(。 ， . ,)
↓
逐条解析
```

**问题:**
- `28.08.38.31.04.各五，07.16.32.19.各十`
- 如果整体部分匹配,可能只识别第一条

---

### 现在的逻辑:
```
策略1: 按"各XX米/蚊"分割 (优先)
   ↓ 成功
返回分割结果

   ↓ 失败
策略2: 按标点符号分割
   ↓
逐条解析
```

**优势:**
- 以"金额结束"为标志,更准确
- 不会因为标点符号误分割
- 支持各种复杂格式

## 💡 核心算法

### splitByAmountPattern 函数

```javascript
// 正则表达式: 匹配"各XX米/蚊/块/元"
const pattern = /(各\s*\d+\s*(?:米|蚊|块|元)?)/g;

// 遍历所有匹配
while ((match = pattern.exec(text)) !== null) {
    // 提取从上一个匹配到当前匹配的文本
    const segment = text.substring(lastEnd, match.index + match[0].length);
    segments.push(segment);
    
    // 跳过分隔符
    while (text[lastEnd] is separator) {
        lastEnd++;
    }
}
```

**关键:**
- 以"各XX单位"为分割标志
- 自动跳过分隔符
- 保留剩余部分
