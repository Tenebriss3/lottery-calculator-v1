# HBuilderX 打包APK完整教程

## 📦 第一步: 下载HBuilderX

1. **访问官网**: https://www.dcloud.io/hbuilderx.html
2. **下载**: 选择"标准版" (免费)
3. **解压**: 解压到任意目录,例如 `E:\HBuilderX`
4. **运行**: 双击 `HBuilderX.exe`

---

## 📝 第二步: 创建项目

1. **打开HBuilderX**

2. **新建项目**:
   - 菜单栏: 文件 → 新建 → 项目
   - 或者快捷键: Ctrl+N

3. **选择项目类型**:
   - 选择 "5+App" 
   - 项目名称: `六合彩工具`
   - 项目路径: 选择 `E:\lottery-betting-tool\hbuilder-project`
   - 点击"创建"

4. **删除默认文件**:
   - 在项目管理器中
   - 删除 `index.html` (默认的)
   - 删除其他默认文件

---

## 📂 第三步: 复制你的文件

1. **打开项目文件夹**:
   - 右键项目 → "在资源管理器中打开"
   - 或者直接打开: `E:\lottery-betting-tool\hbuilder-project`

2. **复制文件**:
   - 从 `E:\lottery-betting-tool\` 复制以下文件/文件夹:
     - `index.html`
     - `css/` 文件夹
     - `js/` 文件夹
   
3. **粘贴到项目**:
   - 粘贴到 `hbuilder-project` 目录
   - 覆盖已有的文件

4. **回到HBuilderX**:
   - 右键项目 → 刷新
   - 应该能看到你的文件了

---

## ⚙️ 第四步: 配置manifest.json

1. **打开manifest.json**:
   - 在项目根目录找到 `manifest.json`
   - 双击打开

2. **基础配置**:
   ```
   应用名称: 六合彩工具
   应用版本: 1.0.0
   应用描述: 六合彩下注管理工具
   ```

3. **图标配置**(可选):
   - 切换到"图标配置"标签
   - 上传APP图标 (512x512 PNG)
   - 或者使用默认图标

4. **权限配置**:
   - 切换到"模块权限配置"
   - 确保勾选:
     - ✓ Storage (存储)
     - ✓ Runtime (运行时)

5. **保存**: Ctrl+S

---

## 🚀 第五步: 云打包APK

1. **右键项目**:
   - 在项目上右键
   - 选择 "发行" → "原生App-云打包"

2. **配置打包参数**:
   ```
   平台: Android
   包名: com.lottery.tool
   版本: 1.0.0
   使用自有证书: 否(使用DCloud公共证书)
   ```

3. **开始打包**:
   - 点击"打包"按钮
   - 等待上传完成
   - 云端编译中... (大约2-5分钟)

4. **下载APK**:
   - 打包完成后会显示下载链接
   - 点击下载APK
   - 保存到电脑

---

## 📱 第六步: 安装到手机

### 方法1: USB传输
1. 手机连接电脑
2. 复制APK到手机
3. 手机打开文件管理器
4. 点击APK安装
5. 允许"未知来源应用"

### 方法2: QQ/微信
1. 发送APK到手机QQ或微信
2. 手机接收文件
3. 点击安装

### 方法3: 网盘
1. 上传APK到百度网盘
2. 手机下载网盘APP
3. 下载并安装

## ⚠️ 常见问题

### Q1: 注册DCloud账号
**A:** 
- 首次使用需要注册
- 访问: https://dev.dcloud.net.cn/
- 免费注册
- 登录后才能云打包

### Q2: 打包失败
**A:**
- 检查网络连接
- 检查manifest.json配置
- 查看控制台错误信息
- 尝试重新打包

### Q3: APP打开空白
**A:**
- 检查index.html路径
- 检查CSS/JS文件是否正确引用
- 在浏览器中先测试是否正常

### Q4: 每天打包次数限制
**A:**
- 免费账号每天有打包次数限制
- 通常足够个人使用
- 如需更多,可以实名认证提升限额
